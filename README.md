# Lost and Found Management System

This is my 1st semester Programming Fundamentals project developed in C++.

## About the Project

The Lost and Found Management System is a simple console-based CRUD application developed in C++.

It helps manage records of lost and found items. The user can create, view, search, update, and delete item records.

This project demonstrates basic CRUD operations: Create, Read, Update, and Delete.

## Features

- Create / add a lost item record
- Read / view all item records
- Search item by name
- Update item details
- Delete item record
- Exit the program

## Concepts Used

- CRUD operations
- C++ structures
- Arrays
- Loops
- Conditional statements
- String input using getline
- Basic search, update, and delete logic

## Technologies Used

- C++
- GitHub
